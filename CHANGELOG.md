# Changelog

## [1.7.0] - 2026-06-11

### Added
- 이미지 첨부 기능: `png`, `jpg`, `jpeg`, `webp`, `gif` 업로드 지원
- 이미지 + 텍스트를 함께 전송하는 Vision API 연동 (gpt-4o 계열)
- 텍스트 없이 이미지만 전송 시 "이 이미지를 분석해줘." 자동 입력
- 채팅 기록에 이미지 썸네일(300px) 인라인 표시
- `gpt-3.5-turbo` 선택 시 이미지 업로더 비활성화 + 안내 메시지

### Changed
- 모델 목록에서 `gpt-5.5`, `gpt-5.5-instant` 제거
- 기본 모델을 `gpt-4o`로 변경

---

## [1.6.0] - 2026-06-11

### Added
- 음성 입력 기능: 마이크 버튼으로 녹음 → OpenAI Whisper(`whisper-1`)로 한국어 음성 인식
- 인식된 텍스트 미리보기 후 자동 채팅 전송
- 마이크 버튼 색상이 현재 테마 accent 색상에 맞게 변경
- 중복 전송 방지 (세션 상태로 마지막 오디오 비교)

---

## [1.5.0] - 2026-06-11

### Added
- 테마별 SVG 배경 이미지: 각 테마에 맞는 패턴을 채팅창 배경에 반투명으로 적용
  - 🌸 봄: 5장 꽃잎 벚꽃 + 떠다니는 꽃잎
  - ☀️ 여름: 사인 파도 + 태양 원 + 물방울
  - 🍂 가을: 기울어진 잎사귀 (주황·빨강)
  - ❄️ 겨울: 눈결정 십자 패턴 + 얼음 점
  - 🌙 밤: 별빛 + 반짝임 + 초승달
  - 🌿 숲: 타원 잎사귀 (초록 계열)
- SVG를 base64 데이터 URI로 내장 (외부 의존 없음)

### Changed
- 사이드바 전체 텍스트 흰색(`#ffffff`)으로 통일

---

## [1.4.0] - 2026-06-11

### Added
- 6가지 계절·분위기 테마: 🌸 봄, ☀️ 여름, 🍂 가을, ❄️ 겨울, 🌙 밤, 🌿 숲
- 테마 선택 드롭다운을 사이드바 상단에 배치

### Changed
- 기본 테마를 `🌙 밤`으로 변경
- 모든 버튼 색상을 WCAG AA 기준(4.5:1 이상) 명도 대비로 수정
  - 🌸 봄 `#f472b6`(2.7:1 ✗) → `#be185d`(5.8:1 ✓)
  - ☀️ 여름 `#0ea5e9`(2.8:1 ✗) → `#0369a1`(5.8:1 ✓)
  - 🍂 가을 `#ea580c`(3.4:1 ✗) → `#9a3412`(7.1:1 ✓)
  - ❄️ 겨울 → `#4338ca`(6.6:1 ✓)
  - 🌙 밤 → `#6d28d9`(6.3:1 ✓)
  - 🌿 숲 → `#166534`(7.2:1 ✓)
- 채팅 버블 내 텍스트에 명시적 색상 적용
- 입력창 플레이스홀더 색상 테마별 적용

## [1.1.0] - 2026-06-11

### Added
- 사이드바 UI: API 키 입력, 모델 선택, 파라미터 조절을 사이드바로 통합
- 모델 선택 드롭다운: `gpt-4o-mini`(기본값), `gpt-4o`, `gpt-3.5-turbo` 중 선택 가능
- Temperature 슬라이더: 0.0 ~ 2.0 범위로 응답 창의성 조절
- 채팅 기록 내보내기: `.txt` 및 `.json` 형식 다운로드 버튼
- 대화 초기화 버튼 (Clear chat)
- API 오류 처리: 키 오류, 네트워크 오류 등 발생 시 화면에 에러 메시지 표시

### Changed
- 기본 모델을 `gpt-3.5-turbo` → `gpt-4o-mini`로 변경
- API 키 입력 위치를 메인 화면 → 사이드바로 이동
